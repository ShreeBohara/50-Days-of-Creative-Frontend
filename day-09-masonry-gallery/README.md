# Day 09 — Masonry Gallery

Infinite scroll masonry gallery with viewport animations, lightbox navigation, and real-time column toggle.

## Tech Used
- CSS Columns (masonry layout)
- Intersection Observer API
- picsum.photos API

## What I Learned
- CSS columns create natural masonry layouts without JavaScript calculations
