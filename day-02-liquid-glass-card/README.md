# Day 02 - Liquid Glass Card

A frosted glass profile card with 3D tilt, a pointer-driven refraction stripe, and an animated gradient border.

## Tech Used

- HTML5
- CSS3
- Vanilla JavaScript
- CSS `backdrop-filter` and custom properties

## What I Learned

Layering blur, specular highlights, and a masked conic-gradient border creates a much more convincing glass effect than blur alone.
