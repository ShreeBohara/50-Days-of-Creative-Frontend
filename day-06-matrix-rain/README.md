# Day 06 — Matrix Rain

Full-screen Matrix digital rain with a hover-activated decode effect that reveals a hidden message letter by letter.

## Tech Used

- HTML5 Canvas API
- Vanilla JavaScript
- CSS3

## Features

- Falling columns of katakana, latin, and numeric characters with fade trails
- Hidden message mapped to grid positions — hover to decode
- Accelerating character cycling that "locks in" like a slot machine
- Decoded characters glow with pulsing animation when fully revealed
- Auto re-encrypt after 3 seconds of mouse inactivity
- CRT scanline overlay and vignette for atmosphere
- Responsive to viewport resizing

## What I Learned

Layering translucent fills to create natural fade trails, and using proximity-based decode timers with accelerating character cycling to make reveals feel physical and satisfying.
