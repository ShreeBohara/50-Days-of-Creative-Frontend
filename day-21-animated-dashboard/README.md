# Day 21 — Animated Analytics Dashboard

A dark-themed analytics dashboard with animated stat cards, live-updating charts, smooth page transitions, and a floating action button.

## Features

- **4 Stat Cards** — Revenue, Users, Orders, Conversion Rate with animated count-up and sparkline mini charts
- **Live Line Chart** — 30 data points that update every 2 seconds with a sliding window
- **Monthly Bar Chart** — Year-over-year comparison with gradient fills
- **Category Donut Chart** — Sales breakdown with animated segments and center total
- **Stacked Area Chart** — Hourly traffic sources with gradient fills
- **Collapsible Sidebar** — 5 navigation pages with animated indicator
- **Page Transitions** — Framer Motion AnimatePresence for smooth view switching
- **Floating Action Button** — Radial menu with 5 quick actions
- **2-Second Live Refresh** — All data updates in real-time with pulse animations
- **Fully Responsive** — Mobile drawer sidebar, stacked cards, single column charts

## Tech Stack

- React 19 + Vite 6
- Framer Motion
- Recharts
- CSS Grid + CSS Custom Properties

## Run Locally

```bash
npm install
npm run dev
```
