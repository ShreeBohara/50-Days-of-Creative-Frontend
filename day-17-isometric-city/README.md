# Day 17 — Isometric City Builder

Canvas-based isometric city builder with placeable tiles, day/night cycle, minimap, and procedural generation.

## Features

- 20x20 isometric grid with 8 tile types
- Buildings rendered as isometric boxes with proper depth sorting
- Smooth day/night transition with window lights
- Corner minimap with real-time updates
- Random city generator
- Touch support and responsive scaling

## Tech

Canvas API, vanilla JavaScript, isometric math
