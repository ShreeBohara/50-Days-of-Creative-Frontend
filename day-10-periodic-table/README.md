# Day 10 — Interactive Periodic Table

A full interactive periodic table with all 118 elements, category color coding, hover tooltips, click modals, search/filter, and a temperature slider showing states of matter.

## Tech Used

- CSS Grid (18-column periodic table layout)
- Vanilla JavaScript (dynamic rendering, event delegation)
- CSS Custom Properties (category color system)

## What I Learned

- CSS Grid with explicit row/column placement handles the periodic table's irregular shape elegantly
- Event delegation on a single container scales well for 118+ interactive elements
