# Day 12 — 3D Card Deck

A premium 3D card deck featuring ten Celestial Arcana cards with cinematic transitions. Stack, fan, shuffle, and flip through the deck using mouse, touch, or keyboard.

## Features

- **3D stacked deck** — ten unique cards with depth offset and perspective
- **Cycle through cards** — click/drag or arrow keys to send cards to the back
- **Fan spread** — spring-animated arc layout revealing all cards
- **Shuffle** — scatter-and-collect with Fisher-Yates reorder
- **Card flip** — 180-degree 3D flip revealing card back details
- **Ambient float** — top card gently hovers
- **Responsive** — scales to mobile with adjusted perspective
- **Keyboard accessible** — full arrow key, Enter, Escape support

## Tech

Vanilla HTML, CSS, JavaScript. Google Fonts (Cinzel Decorative + Cormorant Garamond). CSS 3D transforms with perspective. No frameworks or build tools.
