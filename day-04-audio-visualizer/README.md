# Day 04 - Audio Visualizer Spectrum

A full-screen circular audio spectrum built with Canvas and the Web Audio API, featuring radial FFT bars, beat-reactive pulses, and a built-in demo synth.

## Tech Used

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas API
- Web Audio API

## What I Learned

Driving both uploaded media and a synthetic demo source through the same analyser keeps the rendering pipeline simple while still supporting multiple listening modes.
