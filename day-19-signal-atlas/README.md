# Day 19 — Signal Atlas

Cinematic signal globe with orbitable city hubs, glowing corridors, and a live detail rail that tracks the strongest routes in the network.

## Tech

Canvas API, vanilla JavaScript, orthographic projection, pointer events

## One Thing Learned

Interpolating route paths in 3D before projecting them to the canvas makes globe connections feel anchored to the sphere instead of looking like flat lines pasted on top of it.
